# Accessibility JS File



## Used

To use Accessibily in your website, simply drop the script into your last line document's `<body>`.

```html
<script defer src="https://cdn.jsdelivr.net/gh/alafiAvichai/Accessibility/accessibilityScript.js"></script>
```

look like this

```html
<body>
 
 
 
 
 <script defer src="https://cdn.jsdelivr.net/gh/alafiAvichai/Accessibility/accessibilityScript.js"></script>
 </body>
```
